The Abyss

The abyss is a local-multiplayer platforming-fighting game with no hand to hand combat; instead, you defeat your opponent by outlasting them.

Game and Rules

When the game starts, platforms with different random properties will spawn, and it is now your job to be the last man standing. If you touch the sides of the arena or the bottom of the screen, you're dead. If you touch the dangerous red platforms you're also dead. There are many different kinds of platforms, so try to keep track of what is happening and avoid falling into the abyss.

Development Process

This prototype was made for a university course in a single week by me and a team using Unity. The assignment was to make a game that was spectatable, that is, interesting to watch as well as play. While this was one of the first games I made in Unity, and the game could use a boat-load of polish, I really like the premise of this game. Some day I might return and re-design and re-program this game.
