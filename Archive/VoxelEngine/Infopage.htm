Voxel Engine

This is a Minecraft inspired voxel engine made in Simple Render Engine with C++. Priority was on optimization.

The project

As a final assignement in a university course we were tasked with making a game using the Simple Render Engine (SRE). Our group deciced to create a basic Minecraft-clone, with proper voxel data structure and chunk manangement. The final result is a simple game with the same mechanics as Minecraft: You can mine blocks and place them. We added some small extra features, like flying, a simple particle system and some textures.

What is special about it

The big thing that we acheived was an engine that could run at a stable 60FPS gameplay while 262,144 voxels were loaded. The primary challenges of making our voxel engine was optimisation, mesh generation and getting the data structure right.
